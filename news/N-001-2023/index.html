Flok to be released

Flok, an app that encrypts content of a file using password can help users avoid sharing confidential file and worrying about it going into the wrong hands.



What exactly is flok?

Flok is a tool for anyone looking to encrypt files for any reason whether they're working on a project or just need security from prying eyes. Instead of using an easily detectable file format such as Text files, source codes and etc. TheCrashersTeam designed flok to be fully transparent when encrypting text based files. This allows anyone with basic computer skills to use flok without issue- though it does make decoding encrypted files more difficult.

Because of this design decision, we decided to make flok available as a secure tool for anyone looking to protect their data from prying eyes. Anyone with basic computer skills can download, install, run and use flok with no issues whatsoever. Flok allows users to secure their content of their file with password. Decryption of the file can be done using the flok software only, which means that if data were to go to another user who wasn't aware of the encrypted file, the data will look like a bunch of gibberish.

When will it be released?

Flok is set to release on the 22nd of January, 2023...
